# Autobot app

## Installation

1. Install dotnetcore21
    - choco install dotnetcore
2. Install FAKE (MAKE for .net)
    - dotnet tool install fake-cli -g
3. Install yarn
    - choco install yarn
4. Install node 8
    - choco install nodejs

## Run

from root dir run:

fake build --target run

or

debug in visual studio (/static/index.html)
